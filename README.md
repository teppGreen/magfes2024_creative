# 角川ドワンゴ学園 磁石祭実行委員会(2025) クリエイティブ班 業務システム

##概要
クリエイティブ班というのは、実行委員会が制作する画像やら動画やらを作るチームのことです。
大量の制作依頼・制作物の管理をしやすくするために開発しました。
詳細は説明資料をご覧ください。

- [制作管理シート] 各項目の記入/記録場所の変更
- [制作管理シート] KPTを記入する枠を作成
- その他新機能を追加（詳細: 【説明書】磁石祭2025 クリエイティブ班業務システム ）
